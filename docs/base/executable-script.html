    # 可执行命令

    # npm start

    启动服务，自动打开默认浏览器展示你的页面，通常用于本地开发。

    # npm run build

    编译项目，默认情况下会将文件输出到项目根目录下的 dist 文件夹中

    # npm run build:analyze

    与 npm run build 类似，但会分析项目的依赖包，并在网页中展示分析的结果，通常使用它来优化项目的依赖。

    # npm run lint

    检查并修复代码中出现的格式问题。
